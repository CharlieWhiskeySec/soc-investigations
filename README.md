Security Investigations

## Overview

This repository documents security investigations performed using telemetry generated within my Active Directory and Splunk Enterprise lab environment.

Each investigation is designed to simulate the analytical process performed by security operations teams, beginning with an alert or suspicious event and progressing through evidence collection, timeline reconstruction, and documented findings.

Rather than focusing solely on the final result, these investigations emphasize the reasoning, methodology, and supporting evidence used throughout the analysis.

## Objectives

- Investigate alerts generated by Splunk detections.
- Analyze Windows Security Events and supporting telemetry.
- Document investigation methodology and findings.
- Correlate authentication, endpoint, and network activity.
- Develop repeatable investigation workflows.
- Strengthen incident analysis and documentation skills.
